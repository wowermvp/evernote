目录处理命令

Linux常用命令
命令格式
    命令 [-选项] [ 参数]
    例： ls –a /etc/
    说明：个别命令使用不遵循此格式
          当有多个选项时，可以写在一起
          简化选项与完整选项（如： -a 与 --all ）

命令名称：ls
命令英文原意：list
命令所在路径：/bin/ls
执行权限：所有用户
功能描述：显示目录文件
语法：ls  选项[-ald]  [文件或目录 ]
        -a  显示所有文件，包括隐藏文件（ Linux中以“. ”开头的文件为隐藏文件） all
        -l  详细信息显示  long
        -d  查看目录属性 directory
        -h  人性化显示   human
        -i  查询i节点    inode

   文件权限          所有者        所属组     文件大小     最后一次修改时间   文件名称
 -rw-r--r--.   1    root         root         26150         3 月2 00:28    install.log

   文件的所有者和所属组都只能有一个。 Linux无创建时间概念，只有最后一次修改时间，
   如果文件从创建时起就没修改，最后一次修改时间即创建时间。
       -rw-r—r—
       -      rw-   r—    r—
       - 文件类型（ - 二进制文件 d 目录 l软链接）
       rw-          r--          r—
       u            g           o
       u所有者      g所属组     o 其他人
       r读     w写    x 执行

命令名称：mkdir
命令英文原意：make directory
命令所在路径：/bin/mkdir
执行权限：所有用户
功能描述：创建新目录
语法：mkdir 选项[-p] [目录名]
            -p 递归创建（即可以在创建新目录的同时，在新目录下创建下一级目录,
                        不加-p只能创建一级新目录)
范例：$ mkdir -p /tmp/Japan/boduo           #例子中Japan和boduo都是将要创建的新目录
      $ mkdir /tmp/Japan/cangjing /tmp/Japan/longze   #mkdir可以来同时创建多个目录，
                                                       中间用空格隔开。
注意不要在随便在“/”（根目录）下创建新目录，目录名称要有一定的含义，方便以后查找。

命令名称：cd
命令英文原意：change directory
命令所在路径：shell内置命令
执行权限：所有用户
功能描述：切换目录
语法：cd [目录]
范例：$ cd /tmp/Japan  #切换到指定目录
      $ cd ..         #回到上一级目录

命令名称：pwd
命令英文原意：print working directory
命令所在路径：/bin/pwd
执行权限：所有用户
功能描述：显示当前目录
语法：pwd
范例：$ pwd                 #显示当前所在的目录

命令名称：rmdir
命令英文原意：remove empty directory
命令所在路径：/bin/rmdir
执行权限：所有用户
功能描述：删除空目录
语法：rmdir [目录名]
范例：$ rmdir /tmp/Japan/boduo     #删除/tmp/Japan/boduo这个空目录

命令名称：cp
命令英文原意：copy
命令所在路径：/bin/cp
执行权限：所有用户
功能描述：复制文件或目录
语法：cp -rp [原文件或目录名] [目标目录]
         -r 复制目录（不加-r只能复制文件到指定目录）
         -p 保留文件属性
范例：$ cp -r /tmp/Japan/cangjing /home     #将目录/tmp/Japan/cangjing复制到目录/home下
      $ cp -rp /tmp/Japan/boduo /tmp/Japan/longze /home
                                    #将/tmp/Japan目录下的boduo、longze目录复制到/home下，
                                     保持目录属性cp可以同时复制多个文件，中间以空格隔开。
                                     复制的同时还可以更名。

命令名称：mv
命令英文原意：move
命令所在路径：/bin/mv
执行权限：所有用户
功能描述：剪切文件、改名
语法：mv [原文件或目录] [目标目录]

命令名称：rm
命令英文原意：remove
命令所在路径：/bin/rm
执行权限：所有用户
功能描述：删除文件
语法：rm -rf [文件或目录]
         -r 删除目录
         -f 强制执行（不经询问直接删除）  force
注意删除文件之前做好备份，误删除之后不要做过多的读写操作。
crtl+c用来结束命令
